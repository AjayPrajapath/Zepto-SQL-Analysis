# 🛒 Zepto SQL Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing Zepto's inventory dataset using SQL Server. The goal is to extract business insights related to pricing, discounts, and inventory management.

---

## 🛠️ Tools Used
- SQL Server Management Studio (SSMS)
- SQL (T-SQL)

---

## 📂 Dataset
- Zepto product dataset (CSV file)

---

## 🔍 Key Analysis Performed

1. Top 10 products with highest discount
2. High MRP products that are out of stock
3. Estimated revenue per category
4. Expensive products with low discount
5. Top 5 categories with highest average discount
6. Price per gram analysis for value products
7. Product categorization based on weight
8. Total inventory weight per category

---

## 🧠 Key Insights
- Identified high discount products to attract customers
- Found categories generating maximum revenue
- Detected stock issues in high-value products
- Analyzed product value using price per gram

---

## ⚙️ Data Cleaning Steps
- Removed products with zero price
- Converted price values into INR
- Checked null values and data types
- Added primary key column (sku_id)
